End-to-End SDK Validation

Overview

This test is a full end-to-end validation through the Prebid Mobile SDK to the primary ad server using the Prebid Server configuration.

  • Ad unit registered
    This first step is successful if the Prebid Mobile API call was made to register the ad unit.

  • Request to Prebid Server sent
    This step is successful if a bid request was sent from Prebid Mobile to Prebid Server.

  • Prebid Server response received
    This test passes if a bid response was received from Prebid Server.

  • Creative content cached
    This test passes if the creative content from the bid received from Prebid Server was cached.

  • Ad server request sent and Key-value targeting sent
    Verifies whether the bid request was sent. The request will not be sent if the IDs you entered on the Setup page are incorrect. Tap this entry to see a list of key-value pairs that were sent with this request.
    • Select Prebid Key-Value Pairs to view only the key-values sent that apply to prebid.
    • Select Ad Server Request to view the full list of key-values sent in the ad request.

  • Prebid Mobile creative HTML served
    Tap this entry to view:
    • the Prebid.js HTML code for the Prebid Mobile creative.
    • the creatives that were returned. In a successful test two creatives will be displayed: the creative returned from the ad server and the Prebid.org test creative. Tap the buttons to view each creative.

Results

Possible outcomes from this test:

Success: This is a full, end-to-end validation of your Prebid Mobile implementation. The validation succeeds if every step passes.

Failure: There are many steps throughout this validation. If a problem is encountered at any step, the validation will be marked as failed.
Here are some things to check if this test fails:

  • Check all your settings on the Setup screen. In particular, make sure all the IDs you entered are correct and valid.
  • Make sure you've configured your ad unit registration correctly.
  • Verify that your line items are set up correctly on your ad server.
  • Look through your bid request and bid responses (if any). Check the key-value targeting.
  • If creatives were returned, view the HTML code and the creatives. The Received Creative and Expected Creative should be identical.
